# Phoenix 73 Discord Bot Privacy Policy

Effective date: June 10, 2026

## What the bot collects

Phoenix 73 Discord Bot only processes information needed to run its commands:

- Discord command names and command options you submit.
- Call information entered with `/phoenixcall`, such as units, call type, address, details, cross streets, and channel.
- Announcement text entered with `/phoenixannounce`.
- Basic Discord interaction information, such as the user and server where a command was used.

## How the information is used

The bot uses command data to send calls and announcements to the configured Phoenix 73 / Blackflower CAD endpoints. The bot may also print command activity and errors to the local console or server logs for troubleshooting.

## Where the information goes

Command information is sent to the configured Phoenix 73 / Blackflower CAD server. It may be displayed on station alerting pages, HDTV/table pages, BetaBrite signs, and read aloud by text-to-speech depending on the Phoenix 73 setup.

## Data storage

The bot itself does not use a database. Data may still be stored by Discord, the Blackflower CAD system, Phoenix 73 feed files, web server logs, or system logs on the machine running the bot.

## Data sharing

The bot does not sell personal information. Data is only shared with Discord and the configured Phoenix 73 / Blackflower CAD system needed to provide the bot's functions.

## Data removal

To request removal of bot-related data, contact the Discord server owner or the person operating the Phoenix 73 bot. Some records may also need to be removed from Discord, Blackflower CAD, web server logs, or Phoenix 73 feed files.

## Security

The bot uses a shared token to send data to Phoenix 73 endpoints. Keep the Discord bot token and Phoenix shared token private. Anyone with those tokens may be able to control the bot or send data to Phoenix 73.

## Contact

For privacy questions, contact the Discord server owner or the person operating the Phoenix 73 bot.
